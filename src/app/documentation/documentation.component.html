TFGFinder GEI - Platform Technical Documentation

Overview

TFGFinder GEI is a collaborative platform that connects senior university students, professors, and organizations around Final Degree Projects (FDPs). It helps students find, validate, and get support for their project ideas, and allows organizations to source academic talent to develop needed solutions.

User Roles & Permissions

  • Students: Submit ideas, search opportunities, collaborate with professors or organizations.
  • Professors: View, comment, and offer help on student proposals or organization needs.
  • Organizations: Post real-world problems, match with student ideas or suggest their own.
  • Admins: Manage platform, moderate content, and oversee analytics.

Technology Stack

  • Frontend: Angular
  • Backend:SpringBoot
  • Database: PostgreSQL (relational structure with JSON support)
  • Authentication: OAuth2-based system, managed via TGE's Identity Services
  • Hosting: Cloud-native architecture with CI/CD integration
  • API Layer: RESTful APIs for all platform services

Data Model

  • User: Contains user details, role, and authentication tokens.
  • Project Idea: Stores project title, description, status, and associated user.
  • Organization Need: Contains organization details, problem statement, and contact info.
  • Proposal: Links students to project ideas or organization needs with status tracking.
  • Feedback: Comments from professors or organizations on proposals or ideas.

Process Flows

1. Student-Initiated Project Flow

  • Student submits an idea
  • Professors and organizations browse and offer feedback
  • If interest exists, matching occurs via system

2. Organization-Initiated Project Flow

  • Organization posts a challenge or requirement
  • Students can browse and express interest
  • Matching system allows official collaboration

3. No-Idea Students

  • Browse ideas by others or organizations
  • Get inspired and choose to collaborate
  • Can contact original proposer to join project

Security & Authentication

TFGFinder GEI uses TGE's secure identity services. Each user is authenticated using OAuth 2.0 protocols and is assigned a JWT session token. User roles are enforced at the API and UI level to prevent privilege escalation.
